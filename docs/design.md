- 레이아웃: 좌측 로고 / 우측 네비 링크 (서비스소개, 이용방법)
- 모바일: 로고만 표시

## 검색 영역 (Hero) — juso.go.kr 기준

- 배경: `#152060` (진한 네이비)
- border-radius: `16px`
- padding: `40px 32px`
- 입력창
  - 높이: 60px
  - 배경: `#FFFFFF`
  - border: none, border-radius: `8px`
  - placeholder: `#B0BEC5`
  - 우측 인라인: X(클리어) + 돋보기(검색) 버튼
  - 검색 버튼: 파란 원형 아이콘 버튼
- 검색 결과 카운트: 네이비 배경 위 흰색 텍스트

---

## 결과 행 — juso.go.kr 기준

```
[번호]  [도로명 배지] [한글주소] [복사배지]        [우편번호]  [영문][접기]
        [지번 배지]   [지번주소]
```

- 카드 배경: `#FFFFFF`
- 카드 border: `1px solid #E2E8F0`
- border-radius: `8px`
- 번호: `#6B7280`, 14px

### 배지 스타일
| 배지 | 배경 | 텍스트 | 크기 |
|------|------|--------|------|
| 도로명주소 | `#1B2B6E` | `#FFFFFF` | 12px 600 |
| 지번 | `#E5E8EE` | `#6B7280` | 12px 500 |
| 복사 | `#1B6EBE` | `#FFFFFF` | 12px 500 |

### 우측 액션
- 우편번호: 라벨(12px gray) + 숫자(18px bold)
- 영문 버튼: 아이콘 + 텍스트, 테두리 박스
- 접기 버튼: `— 접기` 텍스트, 파란 배경

### 펼침 영역 (영문주소)
- 배경: `#F5F7FA`
- border-top: `1px solid #E2E8F0`
- 구조: 라벨(60px 고정) + 값 + 복사버튼
